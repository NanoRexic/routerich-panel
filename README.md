# RouteRich Panel

Удобная веб-панель для роутера **Routerich** — всё в одном окне браузера, без лишних настроек.

**Что умеет панель:**
- перезагружать роутер одной кнопкой
- настраивать AmneziaWG (VPN)
- управлять Zapret
- исправлять Opera-Proxy

После установки откройте в браузере адрес роутера с портом **2020**, например: `http://192.168.1.1:2020`

---

## Способ 1 — установка прямо на роутере

Подойдёт, если вы умеете подключаться к роутеру по SSH (через PuTTY, терминал и т.п.).

**Шаг 1.** Подключитесь к роутеру по SSH (логин обычно `root`).

**Шаг 2.** Скопируйте и вставьте одну команду — установка начнётся сама:

```sh
wget -O - https://github.com/NanoRexic/routerich-panel/raw/refs/heads/main/install.sh | sh
```

**Шаг 3.** Когда установка закончится, откройте в браузере:

`http://IP-вашего-роутера:2020`

> Если порт 2020 занят, можно указать другой при установке:
>
> ```sh
> PANEL_PORT=2021 wget -O - https://github.com/NanoRexic/routerich-panel/raw/refs/heads/main/install.sh | sh
> ```

**Удаление панели** — та же схема, другая команда:

```sh
wget -O - https://github.com/NanoRexic/routerich-panel/raw/refs/heads/main/uninstall.sh | sh
```

### Если установка не скачивает файлы

Иногда роутер не может открыть GitHub (например, из-за Zapret). Варианты:

1. **Проще всего** — установите с Windows (способ 2 ниже).
2. Либо добавьте записи в `/etc/hosts`, как в Zapret Manager.

> При ручной установке используйте ссылку через `github.com/.../raw/...`, а не `raw.githubusercontent.com` — на некоторых роутерах вторая ссылка отдаёт старые файлы.

### После обновления панели

Нажмите **Ctrl+F5** в браузере, чтобы страница обновилась полностью.

## Способ 2 — установка с Windows (рекомендуется)

Не нужно разбираться в SSH — достаточно запустить два файла на компьютере.

1. Скачайте архив **RouteRich-Windows.zip** из [Releases](https://github.com/NanoRexic/routerich-panel/releases) и распакуйте.
2. Запустите **`1-Install-Prerequisites.bat`** — один раз, чтобы подготовить компьютер.
3. Запустите **`2-Install-Panel.bat`** — установщик сам подключится к роутеру и всё настроит.

Установщик сам скачает файлы, если роутер не может зайти на GitHub.

**Удаление:** запустите **`3-Uninstall-Panel.bat`**

### Дополнительные параметры

Обычно ничего указывать не нужно. Если роутер нестандартный — можно передать параметры в `2-Install-Panel.bat`:

| Параметр | Зачем нужен |
|----------|-------------|
| `--host IP` | IP роутера, если автоопределение не сработало |
| `--empty-password` | У роутера нет пароля SSH |
| `--password PASS` | Указать пароль SSH сразу в команде |
| `--local-upload` | Скачать файлы на ПК и загрузить на роутер вручную |
| `--test-ssh` | Только проверить, что SSH работает |

## Что нужно для работы

| | |
|---|---|
| Роутер | **Routerich** |
| Доступ | SSH (для установки с роутера) или Windows (для установщика) |
| Порт панели | **2020** по умолчанию (запасные: 2021, 8080, 8888) |
| Веб-сайты роутера | Порты 80 и 443 **не трогаются** — LuCI и сайты работают как раньше |
