# RouteRich Panel

Удобная веб-панель для роутера **Routerich** — всё в одном окне браузера, без лишних настроек.

**Что умеет панель:**
- перезагружать роутер одной кнопкой
- настраивать AmneziaWG (VPN)
- управлять Zapret
- исправлять Opera-Proxy

После установки откройте в браузере: **`http://routerich.lan:2020`**

---

## Способ 1 — установка прямо на роутере

Основной способ. На Routerich уже есть встроенный терминал — откройте **Службы → Терминал** в LuCI, вставьте команду и нажмите Enter.

**Шаг 1.** Откройте терминал в LuCI (**Службы → Терминал**).

**Шаг 2.** Скопируйте и вставьте одну команду — установка начнётся сама:

```sh
wget -O - https://github.com/NanoRexic/routerich-panel/raw/refs/heads/main/install.sh | sh
```

**Шаг 3.** Когда установка закончится, в терминале появится строка вида `Panel URL: http://...:2020/` — **нажмите на неё**, и панель откроется в браузере. Либо откройте вручную: **`http://routerich.lan:2020`**

> Если порт 2020 занят, можно указать другой при установке:
>
> ```sh
> PANEL_PORT=2021 wget -O - https://github.com/NanoRexic/routerich-panel/raw/refs/heads/main/install.sh | sh
> ```

**Удаление панели** — та же схема, другая команда:

```sh
wget -O - https://github.com/NanoRexic/routerich-panel/raw/refs/heads/main/uninstall.sh | sh
```

### Если установка не скачивает файлы

В большинстве случаев установка с роутера проходит штатно. Проблема возможна, если **GitHub заблокирован РКН** (Роскомнадзор) и роутер не может скачать файлы. Тогда можно установить панель через локальный установщик с Windows (способ 2) — он скачает файлы на компьютер и загрузит их на роутер.

> Используйте ссылку через `github.com/.../raw/...`, а не `raw.githubusercontent.com` — вторая ссылка иногда отдаёт устаревшие файлы.

### После обновления панели

Нажмите **Ctrl+F5** в браузере, чтобы страница обновилась полностью.

## Способ 2 — локальный установщик для Windows

Удобен, когда нужно установить панель с компьютера: не открывая терминал на роутере, или если с роутера недоступен GitHub из‑за блокировки РКН. Установщик подключается к Routerich по сети и всё настраивает сам.

1. Скачайте архив **RouteRich-Windows.zip** из [Releases](https://github.com/NanoRexic/routerich-panel/releases) и распакуйте.
2. Запустите **`1-Install-Prerequisites.bat`** — один раз, чтобы подготовить компьютер.
3. Запустите **`2-Install-Panel.bat`** — установщик подключится к роутеру и установит панель.

По умолчанию роутер определяется автоматически. На Routerich можно также указать **`routerich.lan`** — знать IP-адрес шлюза не обязательно.

**Удаление:** запустите **`3-Uninstall-Panel.bat`**

### Дополнительные параметры

Обычно ничего указывать не нужно. При необходимости — параметры для `2-Install-Panel.bat`:

| Параметр | Зачем нужен |
|----------|-------------|
| `--host routerich.lan` | Адрес роутера (по умолчанию определяется автоматически) |
| `--empty-password` | У роутера нет пароля SSH |
| `--password PASS` | Указать пароль SSH сразу в команде |
| `--local-upload` | Скачать файлы на ПК и загрузить на роутер (если GitHub заблокирован РКН с роутера) |
| `--test-ssh` | Только проверить подключение к роутеру |
